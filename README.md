# ContextSystem

コンテキスト（文脈・エージェントやツールに渡す情報の設計など）に関する**知見**と**実験**を置くリポジトリです。

## アーキテクチャの最重要事項（エージェント共通）

Cursor・OpenAI Codex・Claude Code・GitHub Copilot など、**どのエージェントがこのリポジトリをクローンして解析しても**、次を同じ解像度で認識できるように設計する（リポジトリを**ひとつのシステム**として扱う）。

1. **現状、何が使えるか / 使えないか**
2. **使える範囲では、何ができるか**
3. **どうやるか**（手順は文書化されたものに限定）
4. **何が結果か**（期待する出力・成果物）

そのための**正本はルートの `AGENTS.md` のみ**（ツールごとの設定ファイルは置かない）。人間向けの概要は本 README、可否・手順・結果・更新ルールはすべて `AGENTS.md` に集約する。  
各 AI エージェントは、**セッションごとに `AGENTS.md` をコンテキストに含める・開いて読む・プロンプトで参照させる**など、ツール中立の方法で同じファイルを読ませること。

## 設計原則

- **ルート直下の各ディレクトリ**がひとつの単位です。
- 各ディレクトリは**情報スコープがそこで閉じる**ようにします（他ディレクトリの前提を読まなくてよい）。
- 各ディレクトリは**単体で利用できる**ようにドキュメント・コード・設定を揃えます。

トップレベルに新しい試みやノートを追加するときは、この原則に沿って切り出し、**`AGENTS.md` の「ルート直下スコープ一覧」と各スコープ内 `AGENTS.md`** を更新してください。
