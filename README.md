# ContextSystem

コンテキスト（文脈・エージェントやツールに渡す情報の設計など）に関する**知見**と**実験**を置くリポジトリです。

## ドキュメントの置き場

- **人間向けの入口**: 本 README（この節まで）。
- **エージェント向けの仕様本文（正）**: ルートの **`CONTEXTSYSTEM_SPEC.md`**（可否・手順・結果・ディレクトリ単位の原則・更新ルールはすべてここだけに書く）。
- **ツールが慣習的に拾う入口**: ルートの **`AGENTS.md`**（本文は持たず、`CONTEXTSYSTEM_SPEC.md` へのリンクのみ）。

トップレベルに試みやノートを追加するときは、`CONTEXTSYSTEM_SPEC.md` の「ルート直下スコープ一覧」と、各スコープ内の `AGENTS.md` を更新してください。
